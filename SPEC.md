## 概要

**サービス名**: Tax Bracket Calculator
**バージョン**: 2.0.0
**更新日**: 2026-05-29
**URL**: https://appadaycreator.com/tax-bracket-calculator/

年収・控除額から所得税・住民税・社会保険料の実効税率と年税額を計算。手取り額の目安も表示。登録不要・完全無料でご利用いただけます。

## 計算ロジック

### 会社員の場合

```
給与所得控除 = 収入金額に応じた段階的控除（〜195万上限）
社会保険料 = 健保(4.99%) + 介護(40歳以上0.91%) + 厚生年金(9.15%) + 雇用保険(0.6%)
              ※協会けんぽ東京都2024年度料率。健保上限139万/月、厚生年金上限65万/月
課税所得（所得税）= 年収 - 給与所得控除 - 基礎控除(48万) - 社会保険料 - 配偶者扶養控除 - その他
課税所得（住民税）= 年収 - 給与所得控除 - 基礎控除(43万) - 社会保険料 - 家族控除(各33万) - その他
所得税 = 超過累進課税（5%〜45%の7段階）× 復興特別所得税(2.1%)
住民税 = 課税所得(住民税用) × 10% + 均等割5,000円
手取り = 年収 - 所得税 - 住民税 - 社会保険料
```

### フリーランスの場合

```
給与所得控除 = なし（代わりに青色申告特別控除 65万円）
社会保険料 = 国民年金(203,760円) + 国民健康保険(概算: 所得割9.37% + 均等割50,000円、上限870,000円)
              ※40〜64歳は介護分(所得割+2.8%)加算
課税所得以降の計算は会社員と同様
```

## 使い方

1. スライダーまたは入力欄に年収（万円）を入力する
2. 雇用形態（会社員 / フリーランス）と年齢を選ぶ
3. 配偶者控除・扶養親族数・iDeCoなど追加控除を入力する
4. 「税額を計算する」ボタンを押して結果を確認する
5. 棒グラフ・内訳テーブルで税負担を把握し、条件を変えて比較する

## よくある質問（FAQ）

**Q: 実効税率とは何ですか？**
年収全体に対して実際に支払った税金（所得税＋住民税）の割合です。累進課税では高所得ほど高い税率が適用されますが、実効税率は全体の平均的な税負担率を示します。

**Q: 社会保険料は税金ですか？**
厳密には税金ではありませんが、給与から天引きされる準税金的な負担です。会社員の場合は健保・厚生年金・雇用保険で年収の約14〜16%になります。

**Q: 住民税はいつ納めますか？**
住民税は前年の所得に基づき翌年6月〜翌々年5月に納めます。会社員は給与から月々天引き（特別徴収）、フリーランスは年4回の納付書払いが一般的です。

**Q: フリーランスと会社員で税額はどう違いますか？**
フリーランスは給与所得控除が適用されない代わりに経費計上ができます。また、社会保険は国民健康保険・国民年金となり、同収入でも保険料が会社員より高くなる場合があります。

**Q: 入力したデータはどこかに保存されますか？**
いいえ。すべての計算はブラウザ内で処理され、データのサーバー送信は行いません。
